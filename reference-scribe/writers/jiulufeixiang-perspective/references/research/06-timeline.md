## 完整时间线

### 奠基期（1992–2013）：重庆成长与网文起步

| 时间 | 事件 |
|------|------|
| 1992年 | 出生于重庆九龙坡，成长于建设厂家属区，自称"厂三代"。童年受丰都"鬼城文化"与"凤凰文化"熏陶 |
| ~2010年 | 大学期间（西华大学艺术设计专业）在晋江文学城注册账号开始创作，初衷是"圆写作梦，顺便赚零花钱" |
| 2010-2013年 | 创作《忘川劫》《司命》《祥云朵朵当空飘》等多部仙侠小说，完成短篇《陈先生与程太太》 |
| 2013年 | 创作完成仙侠小说《本王在此》（后更名《与凤行》） |

---

### 成长期（2014–2018）：首部出版与全职创作

| 时间 | 事件 |
|------|------|
| 2014年 | 《本王在此》正式出版，大学毕业。短暂工作十余天后辞职，成为全职作者 |
| 2016年 | 《一时冲动，七世不祥》（后改编为《七时吉祥》）售出网剧版权，获七位数收入 |
| 2018年 | 在网易云阅读平台连载《驭妖》（出版名《驭鲛记》） |

---

### 爆发期（2019–2023）：仙侠IP"影视化女王"

| 时间 | 事件 |
|------|------|
| 2019年 | 成为恒星引力签约创作者；首部担任编剧的仙侠剧《招摇》播出 |
| 2021年6月3日 | 加入中国作家协会，完成从网络作家到专业作家的身份跨越 |
| 2022年 | 影视改编"丰收年"：《与君初相识》（《驭鲛记》）、《苍兰诀》（《魔尊》）、《护心》接连播出 |
| 2023年8月10日 | 《七时吉祥》（《一时冲动，七世不祥》）播出 |

---

### 成熟期（2024–2026）：多线开发与创作转向

| 时间 | 事件 |
|------|------|
| 2024年3月 | 担任编剧的《与凤行》在湖南卫视播出，市场反响热烈 |
| 2024年11月 | 携短篇小说集《师父系列》亮相泰国曼谷书展 |
| 2025年1月 | 新版《师父心塞》由湖南文艺出版社再版 |
| 2025年4月 | 《来战》官宣定档爱奇艺与腾讯视频 |
| 2025年7月 | 《苍兰诀2》在爱奇艺播出 |
| 2025年 | 创作转向：完成离婚题材现代言情小说《将将好》，标志从仙侠向现实题材拓展 |
| 2026年3月 | 以重庆市网络作协副主席身份出席阅文IP盛典，探讨AI对网文IP开发的影响 |
